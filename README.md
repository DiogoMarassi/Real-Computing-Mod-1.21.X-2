  Um mod para Minecraft 1.21.X (Fabric) que adiciona computação lógica programável ao jogo através de blocos de redstone avançados.

  Descrição

  O Real Computing Mod traz componentes de lógica digital para o Minecraft, permitindo que você construa circuitos lógicos complexos de forma mais compacta e intuitiva. O destaque é o Magic Block, um bloco de lógica programável que pode executar operações lógicas fundamentais.

  Recursos

  Magic Block (Bloco Mágico)

  O coração do mod - um bloco de lógica programável com interface gráfica interativa.

  Características:
  - 4 Operações Lógicas:
    - NOT - Inverte o sinal de entrada A
    - AND - Retorna verdadeiro se A E B estiverem ativos
    - OR - Retorna verdadeiro se A OU B estiverem ativos
    - XOR - Retorna verdadeiro se apenas um dos inputs estiver ativo
  - Inputs/Outputs de Redstone:
    - Input A: Lado esquerdo (WEST relativo ao facing)
    - Input B: Lado direito (EAST relativo ao facing)
    - Output: Lado traseiro (oposto ao facing)
  - Interface Gráfica:
    - Clique com o botão direito no bloco para abrir a GUI
    - Selecione a operação desejada clicando nos botões
    - A operação atual é destacada visualmente
  - Efeitos Visuais:
    - Emite luz baseada em quantos lados estão ativos (máximo 15)
    - Partículas de redstone vermelhas aparecem nos lados ativos
    - Som de sino ao interagir com o bloco
  - Comportamento Direcional:
    - Se orienta automaticamente ao ser colocado (oposto ao jogador)
    - Os lados de input/output são relativos à rotação do bloco


  Uso do Magic Block

  1. Colocação: Coloque o Magic Block no mundo - ele se orienta automaticamente
  2. Configuração: Clique com o botão direito para abrir a GUI e selecionar a operação
  3. Conexão: Conecte sinais de redstone aos lados corretos:
    - Esquerda (A) e Direita (B) para inputs
    - Trás para output
  4. Construção: Use múltiplos blocos para criar circuitos lógicos complexos

  Desenvolvimento

  Este mod foi desenvolvido como um projeto educacional para demonstrar:
  - Criação de blocos customizados com Block Entities
  - Implementação de GUIs com networking cliente-servidor
  - Sistema de redstone customizado
  - Sincronização de estado entre cliente e servidor
  - Efeitos visuais (partículas e iluminação dinâmica)
